# CLAUDE.md

## Style

- No em dashes. Use commas, colons, semicolons, or periods instead.

## Memory

- Do not use the auto memory system (~/.claude/projects/.../memory/). Store any notes or context in .md files within this repo instead.

## Data

- `data/comedians.js` is for comics who have already performed at a show. Do not add new comics there until after they have performed.
- When adding performers to an upcoming show, only update `data/shows.js`.
